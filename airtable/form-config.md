# Airtable Form Configuration

This document specifies the fields and configuration for the Project Intake Form.

## Existing Fields (No Changes Needed)

These fields already exist in the Projects table:

| Field Name | Type | Notes |
|------------|------|-------|
| Project | Single line text | Primary field - project title |
| Project Acronym | Single line text | Short code |
| Start Date | Date | |
| End Date | Date | |
| Status | Single select | In Ideation, Active, On Ice, Completed, Cancelled |
| Asana Board | URL | Already exists |
| SOW | Long text | Already exists - will store generated scoping doc |

## New Fields to Add

### Projects Table - Add These Fields

| Field Name | Type | Configuration |
|------------|------|---------------|
| Project Description | Long text | For detailed project overview |
| Milestones | Linked records | Link to new Milestones table |
| Milestone Summary | Rollup | See config below |
| Roles Summary | Rollup | See config below |

### New Milestones Table

Create a new table called **Milestones**:

| Field Name | Type | Notes |
|------------|------|-------|
| Milestone Name | Single line text | Primary field |
| Project | Linked records | Links to Projects table |
| Due Date | Date | Target completion |
| Description | Long text | Optional details |
| Status | Single select | Not Started, In Progress, Complete |

### Assignments Table - Add This Field

| Field Name | Type | Formula |
|------------|------|---------|
| Role Display | Formula | `{Role} & ": " & {Data Team Member}` |

## Rollup Field Configurations

### Milestone Summary (in Projects)

- **Field type**: Rollup
- **Linked record field**: Milestones
- **Field to rollup**: Milestone Name
- **Aggregation**: `ARRAYJOIN(values, "\n")`

### Roles Summary (in Projects)

- **Field type**: Rollup
- **Linked record field**: Assignments (reverse link)
- **Field to rollup**: Role Display (the formula field)
- **Aggregation**: `ARRAYJOIN(values, "\n")`
