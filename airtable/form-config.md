## Form Configuration

### Fields to Include on Form

| Field | Required | Notes |
|-------|----------|-------|
| Project | Yes | Project title |
| Project Acronym | No | Short code |
| Project Description | Yes | Detailed overview |
| Start Date | No | |
| End Date | No | |

### Fields NOT on Form

| Field | Reason |
|-------|--------|
| Status | Defaults to "In Ideation" |
| Milestones | Added after via linked records |
| Asana Board | User adds after creating board |
| Scope of Work - Generated | Auto-generated by automation |
| Roles Summary | Derived from Assignments |
| Milestone Rollup | Derived from Milestones |

## Status Options

| Status | Description |
|--------|-------------|
| In Ideation | Initial concept, not yet started |
| Active | Currently in progress |
| On Ice | Paused or deprioritized |
| Completed | Successfully finished |
| Cancelled | Discontinued |
