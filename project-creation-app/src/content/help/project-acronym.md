# Project Acronym

A short code or abbreviation for your project, useful for quick reference in file names, folders, and communication.

## Format Suggestions

- **3-6 characters** is ideal
- Use **uppercase letters** for consistency
- Consider including the **year** for annual projects
- Avoid special characters

## Examples

| Project Name | Acronym |
|--------------|---------|
| Climate Data Analysis Initiative | CDAI |
| 2025 Annual Impact Report | AIR25 |
| Water Quality Dashboard | WQD |
| Community Health Assessment | CHA |

## Where It's Used

- Google Drive folder names: "CDAI - Project Documents"
- Document titles: "CDAI - Scoping Document"
- Asana project tags
- Internal references

- Check that the acronym isn't already in use
